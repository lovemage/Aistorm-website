4. 支付成功通知问题排查

可能的问题原因：
  • 🔧 Telegram Bot Token 或 Chat ID 配置错误
  • 🌐 OxaPay webhook URL 配置错误 (应为: https://www.aistorm.art/oxapay-webhook)
  • 🔐 OxaPay 签名验证失败
  • 📡 网络连接问题
  • 🐛 webhook 接收到的数据格式不正确

检查步骤：

  1. 确认 Telegram Bot 配置正确 (上方测试 1)
  2. 确认能发送测试消息 (上方测试 2)
  3. 检查 OxaPay 后台 webhook URL 设置
  4. 查看服务器日志中的 webhook 接收记录
  5. 验证支付确实成功并触发了 webhook

5. OxaPay Webhook 数据格式

OxaPay 支付成功时应发送的数据格式：

{ "orderId": "你的订单号", "trackId": "OxaPay追踪ID", "status": "Paid" 或 "Completed", "amount": "支付金额", "currency": "USDT", "sign": "签名(可选)" }
